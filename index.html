視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，網站背景的移動速度比前景慢，讓使用者在捲動頁面時產生 3D 效果，增加深度並創造更沉浸的瀏覽體驗。視差源自光學錯覺，因為人眼會將較近的物體看得比遠處更大，也因此會覺得遠方的物體移動得較慢。這種錯覺在各種媒介中被應用，形塑出逼真的效果。 最早可以追溯到迪士尼《白雪公主與七矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。隨著 CSS 與 HTML 的發展，視差效果也逐漸在我們熟悉的網頁設計中成形。

2007 年，網頁開發者 Glutnix 在部落格分享了最早的瀏覽器視差滾動實作之一，提供支援 IE6 與其他瀏覽器的 JavaScript 與 CSS2 範例與示範。不過直到 2011 年，隨著 HTML5 與 CSS3 的普及，這項技術才真正受到設計師歡迎。支持者認為這是擁抱網路流動性的簡單方式，並運用視差背景來吸引用戶、提升網站體驗。然而 2013 年普渡大學的研究指出：「……雖然視差滾動強化了使用者體驗的特定面向，但不一定能改善整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 有助於說故事

    視差圖像非常適合用來講故事。好的設計就是有效溝通——在設計網站時，我們希望與訪客交流，並講述產品、服務或品牌的吸引人故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至能透過單頁式網站讓使用者不需切換頁面，就能閱讀完整的品牌或產品故事。

  • 提升使用者參與度

    視差圖像也能提高使用者參與度。降低網站跳出率是設計時需要注意的目標之一，跳出率衡量的是訪客只看一頁就按下「返回」或關閉分頁的比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的作法都值得嘗試。精心打造的視差頁面能帶來動態體驗，讓使用者更願意停留。


如果想在網站加入視差滾動效果，請考慮以下技巧。

  • 選擇性地使用

    將視差滾動用在特定元素上會比在每個頁面、每個區塊都使用更有效果。可以放在頁首或主標題，或僅限首頁。目標是強化網站設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差滾動會使用多種媒體檔案與 CSS 形狀來營造深度。不要使用過大的圖片檔案，避免拖慢載入速度、影響效能與體驗。

  • 玩味色彩

    視差滾動不僅關乎前景與背景圖像，也涉及色彩。背景可使用較低彩度的色調，前景則採用更鮮明的顏色，營造更強烈的深度感。
